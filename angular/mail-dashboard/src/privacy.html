MailNotifier

Politique de confidentialité

Dernière mise à jour : avril 2026

1. Introduction

MailNotifier est une application de surveillance d'e-mails qui envoie des notifications sur Telegram et WhatsApp lorsque vous recevez de nouveaux messages. Cette politique explique comment nous collectons, utilisons et protégeons vos données.

2. Données collectées

  • Compte Google / Gmail : via OAuth 2.0, nous accédons à votre boîte de réception en lecture seule pour détecter les nouveaux e-mails.
  • Informations de profil : nom, adresse e-mail principale.
  • Identifiants de notification : Telegram Chat ID, numéro de téléphone WhatsApp.
  • Tokens OAuth : access token et refresh token Google, stockés de façon sécurisée en base de données.

3. Utilisation des données

  • Détecter les nouveaux e-mails dans votre boîte Gmail.
  • Envoyer des notifications sur vos canaux configurés (Telegram, WhatsApp).
  • Afficher le contenu de vos e-mails dans le dashboard.

Nous n'utilisons pas vos données à des fins publicitaires et ne les partageons avec aucun tiers.

4. Accès Gmail (Google API)

L'application utilise le scope gmail.readonly — accès en lecture seule. Nous ne pouvons pas envoyer, supprimer ou modifier vos e-mails. L'accès peut être révoqué à tout moment depuis votre compte Google : myaccount.google.com/permissions.

5. Stockage et sécurité

  • Les données sont stockées dans une base de données PostgreSQL sécurisée (chiffrement en transit SSL).
  • Les tokens OAuth sont stockés côté serveur uniquement et jamais exposés au navigateur.
  • Les mots de passe sont hachés avec bcrypt.

6. Conservation des données

Vos données sont conservées tant que votre compte est actif. Vous pouvez demander la suppression de votre compte et de toutes vos données à tout moment en nous contactant.

7. Vos droits

  • Accéder à vos données personnelles.
  • Demander la rectification ou la suppression de vos données.
  • Révoquer l'accès Gmail à tout moment.
  • Déconnecter votre compte depuis le dashboard.
